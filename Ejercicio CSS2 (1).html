HTML

HTML siglas en inglés de HyperText Markup Lenguage hace referencia al lenguaje de marcado para a elaboración de páginas web,no confundir con un lenguaje de programacion.Es un estandar a cargo del World Wide Web Consortium(W3C) organización dedicada a la estandarización de casi todas las tecnologias ligadas a la web

Primeras especificaciones HTML

Tim Bernes lee1 en 1991 describe 18 elementos que inclusen el diseño inicial y relativamente simple de HTML. Bernes lee consideaba el HTML una ampliación de SGML, pero no fue formalmente reconocida como tal hasta la publicacion a mediados de 1993 por la IETF, de un primera preposición para una especificacion del HTML: el borrador del HyperText Markub Lenguage de Bernes-lee y Dan Connolly

Marcador HTML

El HTML se escribe en forma de <etiquetas >, rodeadas por corchetes angulares(< > /).El HTML tambien puede describir hasta cierto punto la aparencia de un documento y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede afectar el comportamiento de navegadores web.
